jquery.scrollbar
================

summery: jquery plugin for vertical/horizontal scrollbar

main fetures:
1. the scrollbar doesn't change/rip the container who is operate on
2. the scrollbar location is completely dynamic
3. supports virtual scrolling - e.g. display/hide elements while scrolling, 
   usefull for saving browser's memory usage on large list of items / grid / scroll area
4. supports multiple containers scrolling with one scrollbar (for example - freezing header of columns / rows)
